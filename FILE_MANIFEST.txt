GETTING STARTED
================================================================================

1. Open in browser:
   - Simple: Open index.html in browser
   - Server: python -m http.server 8000 (Python 3)
   - Server: npx http-server (Node.js)

2. Test accessibility:
   - Install Axe DevTools
   - Install WAVE extension
   - Use Lighthouse in Chrome DevTools
   - Test with keyboard (Tab, Enter, Escape)
   - Test with screen reader

3. Customize:
   - Edit HTML files with your info
   - Modify colors in styles.css
   - Add your projects to projects.html
   - Update contact form handler

4. Deploy:
   - Upload to web server
   - No build process needed
   - No dependencies required
   - Works anywhere
